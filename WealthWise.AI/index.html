API Documentation

Financial Advice API

Endpoint: POST /api/financial-advice

Description: Generate AI-powered financial advice based on user profile

Request Body:
{ "age": 30, "income": 75000, "savings": 50000, "risk_tolerance": "moderate", "investment_goals": ["retirement", "house"], "investment_horizon": 20 }
Response:
{ "success": true, "advice": { "title": "Investment Strategy", "description": "Detailed advice...", "action_items": [...], "projected_impact": 125000, "confidence_score": 0.85 } }
Sentiment Analysis API

Endpoint: POST /api/sentiment-recommendations

Description: Analyze market sentiment with AI-powered recommendations

Request Body:
{ "symbols": ["AAPL", "MSFT"], "news_sources": ["Reuters"], "time_period": "1week", "analysis_type": "comprehensive" }
Response:
{ "success": true, "recommendations": [{ "symbol": "AAPL", "sentiment": "Positive", "confidence": 0.82, "recommendation": "Buy" }] }
Action Plan API

Endpoint: POST /api/action-plan

Description: Generate detailed action plans for financial goals

Request Body:
{ "current_portfolio": { "stocks": 60000, "bonds": 30000 }, "financial_goals": [{ "name": "retirement", "target_amount": 1000000, "time_horizon": 25 }], "monthly_savings": 2000 }
Response:
{ "success": true, "action_plan": { "action_items": [...], "estimated_timeline": "6-12 months", "priority_summary": "3 high, 2 medium" } }
